# alQa Brands Creatives

A modern, responsive business website for a creative agency offering design, branding, and printing services. Built with HTML, CSS, and JavaScript.

![Website Screenshot](images/Nairobi.webp) <!-- Replace with an actual screenshot path -->

## Features

- Fully responsive design (mobile, tablet, desktop)
- Interactive laptop scene with animated popping icons
- Three‑section carousels (Design, Branding, Printing) with 32 images each – slides horizontally, images touch edge‑to‑edge
- Live quote builder – add items with quantity and notes, stored in browser
- WhatsApp & email integration – send inquiry with form data + quote items
- Theme switcher (dark/light mode)
- Sidebar navigation with dropdown
- Client testimonials
- About Us section with company stats and location

## Technologies Used

- HTML5
- CSS3 (custom properties, Flexbox, Grid, keyframe animations)
- Vanilla JavaScript (ES6)
- Google Fonts (Inter)
- LocalStorage for cart persistence
- Git & GitHub for version control
